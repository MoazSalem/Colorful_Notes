# ** Notes **

A Clean looking modren and colorful notes app using a local database with sqflite.

Now with 2.0.0 Update</br>

# ** Features **

### - Modern Design</br>

### - Colorful Notes</br>

### - Search Notes</br>

### - Edit and Remove Notes</br>

### - 3 View Styles</br>

### - Tablet Suppoet</br>
