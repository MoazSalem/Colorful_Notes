# Colorful Notes

A Clean Looking Modern and Colorful Notes App Using a Local Database with Sqflite.

# Features

### - Material 3 Theming With Dynamic Colors</br>

### - Both Text and Voice Notes</br>

### - Search Notes & Edit Notes</br>

### - Supports Arabic</br>

### - 3 View Styles</br>

### - Tablet Support</br>
